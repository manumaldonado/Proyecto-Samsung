Informe: Análisis del Poder Adquisitivo y Patrones de Consumo de los Jóvenes Venezolanos

1. Introducción

Este proyecto tiene como objetivo ayudar a los jóvenes a tomar mejores decisiones financieras mediante una herramienta interactiva y análisis de datos. Nuestro objetivo será mejorar la educación financiera y la toma de decisiones a través de este enfoque social y financiero.

2. Metodología

Planificación y Gestión

Se emplearon metodologías ágiles con enfoque en Kanban para dividir las tareas y organizar el trabajo. Además, se utilizó Git y GitHub para el manejo colaborativo del proyecto y control de versiones.

Obtención de Datos

Los datos fueron recolectados a través de formularios de Google Forms, donde el equipo definió las variables clave, tales como ingresos, ahorros y gastos.

Análisis de Datos

Con Python, se procesaron y limpiaron los datos mediante las librerías Pandas y NumPy. Posteriormente, se generaron gráficos y visualizaciones utilizando Matplotlib y Seaborn para identificar patrones significativos.

3. Desarrollo

Herramientas Empleadas

Se implementó la calculadora financiera y se utilizó GitHub para sincronizar los cambios y mantener revisiones organizadas. El análisis de datos se realizó con las librerías antes mencionadas.

Visualizaciones

Los gráficos creados con Matplotlib y Seaborn permitieron representar de manera clara y efectiva los patrones encontrados en los datos recolectados.

Página Web

La página web se diseñó para presentar los resultados de manera interactiva y accesible. Además, se integró la calculadora financiera como una funcionalidad clave.

4. Resultados

Se obtuvieron visualizaciones claras sobre el comportamiento financiero de los jóvenes. La calculadora se presentó como una herramienta funcional, y el sitio web proporcionó una plataforma para acceder a todos los resultados.

5. Conclusiones

Este proyecto permitió mejorar la comprensión de las condiciones financieras de los jóvenes, mientras que el equipo adquirió experiencia en análisis de datos, metodologías ágiles y desarrollo de aplicaciones web.
